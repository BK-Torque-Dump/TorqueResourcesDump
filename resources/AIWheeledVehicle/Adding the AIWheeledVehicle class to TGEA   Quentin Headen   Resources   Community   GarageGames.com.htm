Adding the AIWheeledVehicle class to TGEA

by Quentin Headen · 02/23/2009 (2:16 pm) · 8 comments

I stumbled across this problem while porting my work in progress racing game from TGE to TGEA. TGEA does not have an AIWheeledVehicle class. I would like to first thank Matt Fairfax (GG Employee) for helping me with some stuff in the IRC chat.

Here we go.

First open up vehicle.h in folder T3D/vehicles. Right under the line of code that says void renderMountedImage(SceneState *state, RenderInst *ri); (it should be around line 220) add this line of code:

view plainprint?
  1. virtual bool getAIMove(Move* move);  
virtual bool getAIMove(Move* move);

Now open vehicle.cpp in the same folder. Locate the function Vehicle::ProcessTick(const Move* move).

At the very top of this function add:
view plainprint?
  1. // if we're not being controlled by a client, let the   
  2.    // AI sub-module get a chance at producing a move  
  3.    Move aiMove(NullMove);  
  4.    if(isServerObject() && getAIMove(&aiMove))  
  5.       move = &aiMove;  
// if we're not being controlled by a client, let the 
   // AI sub-module get a chance at producing a move
   Move aiMove(NullMove);
   if(isServerObject() && getAIMove(&aiMove))
      move = &aiMove;

Now at the very bottom of this file add:
view plainprint?
  1. bool Vehicle::getAIMove(Move *move)  
  2. {  
  3.    return false;  
  4. }  
bool Vehicle::getAIMove(Move *move)
{
   return false;
}

Ok. Now download and extract these 2 files from here. They are modified versions of the aiWheeledVehicle code from TGE. They are modded to work with TGEA.

NOTE: For those using TGEA 1.7.0 (and I would guess 1.7.1 also) you have to make a couple of changes to the aiWheeledVehicle.cpp file.

Change 1) Change the include at line 7 from #include "math/mPoint3.h" to #include "math/mPoint.h"

Change 2) Change the line of code at line 100 from front.y=box.maxExtents.y to front.y=box.max.y;


Extract them to the TGEA source folder T3D. Now add them into the project of the engine source code. After that, recompile the engine. You will now have AIWheeledVehicle functionality with TGEA. Enjoy :)
