Improved AI Gaurd ported to T3D

by Justin P Greer · 08/28/2010 (12:27 pm) · 8 comments

Download Code Files
INSTALLATION

These instructions are MODIFIED instructions of the TGE 1.5.2 Documentation found at
www.torquepowered.com/community/resource/view/15320/

I had to look through the Script and port it to work with T3D which was not hard really just time consuming because I had no idea what I was doing! So after a few hours messing with it here is the modified instructions for installing Twisted Jenius's free resource. I do not take credit for the resource, I just modified it to work with T3D. Figured I could put this up as a UPDATED resource, to stop a lot of un need head banging.

I am going to explain what I did just so beginners can kinda get an idea of how the format changed.
The First step in installing IG ( Improved Gaurd ) was to go to the game.cs file and edit it. It said to
replace with but I think he meant was to add a line after. The new file structure for T3D this step was found in

GameCore.cs.
Now there is a Game.cs patch to make T3d more like TGE but I found that I liked it this way. If
you feel like you want to add the patched file setup then do a search for it on the resources. Sorry I can't remember where I seen it.

3. Replace your old aiPlayer.cs with the new one and just place aiPlayerDatablocks.cs in the same folder.
4. Load your Level
5. Go into the map editor (F11). Then go to Object Editor
6. Under Shapes there should be a new folder called AIMarker, under that a new item called AIPlayerMarker.
7. Create a new AIPlayerMarker.
8. Select your marker, position it where you like it
9. You can edit the defaulkt values by checking out the the Dynamic_Setting Doc.txt file
10. Once you have the marker in place save and restart Torque and the effects have taken effect.


Other helpful NOTES
The marker works off of a DTS file that is not included in the resource but is easy enough the edit to your own model/marker. Open the aiplayer.cs file and you can file the line that you need to change around line 87 or so.
I have commented it some more so you can find it. ( Do a search for .dts and it should take you to the line ). Without the model it will not work.

- I did already place the updated bug fix by twisted in these file

The Updates file here have been updated to work with T3D with ease. The changes that need to be changed were mostly Default Datablocks from TGE and not used in T3D. Mainly just DefaultPlayer to DemoPlayerchanges were made. The port was very easy but for a beginner it may have made you over look this GREAT resource.


I suggest that you look into the AI Marker Editor also created by Twisted found here:
[url]http://www.torquepowered.com/community/resources/view/16075 [/url]

The Ai marker GUI too does needed changed a little to make it work and I will post a updated resource also upon request.
