afxFPS - T3D Beta & AFX 2.0

by Sean Rice · 03/12/2010 (6:13 am) · 21 comments

www.filefront.com/17287011/afxFPS-ORIG-TGEA-T3D.rar

Originally the work of Gibby at www.torquepowered.com/community/resources/view/17097 and with my first conversion for TGEA here. After a couple of people asked for updated versions, I threw one together here.

Corrected all afxFPS paths to work with AFX 2.0 on T3D Beta 1.1 and use only AFXDemo supplied items.
Minor tweaks to the scripts, completed separation of player datablocks.
Changed Third person camera to not be glued to the players head at a specific angle. This was driving me crazy and was not allowing true 3rd person viewing.
Changed camera toggle to
Toggle 1 - Casting mode third person
Toggle 2 - Combat mode first person
Toggle 3 - Casting mode first person
Toggle 4 - Combat mode third person

function serverCmdForceModaPOV(%client)
{
  // get 3rd-person camera offset 
  %cam_dist = %client.camera.getThirdPersonDistance();

  // save 3rd-person distance, force first-person on
  if (%cam_dist <= 1.0)
  {
    %client.save_3rdperson_dist = %cam_dist;
    %cam_dist = 3.0;
  }
  // restore 3rd-person distance, force first-person off
  else if (%client.save_3rdperson_dist !$= "")
  {
    %cam_dist = (%client.save_3rdperson_dist !$= "") ? %client.save_3rdperson_dist : 3;
  }

  // set new 3rd-person distance
  %client.camera.setThirdPersonDistance(%cam_dist);
  %client.camera.setThirdPersonSnap();
}
